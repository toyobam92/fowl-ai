# FOWL AI Newsletter Site

Upload this folder to any static host.

Start page:

```text
index.html
```

Issue archive:

```text
issues/2026-06-03/index.html
```

For each new week:

1. Create a new folder inside `issues/`, for example `issues/2026-06-10/`.
2. Add that week's newsletter as `index.html`.
3. Add a new row to the archive list in `index.html`.
